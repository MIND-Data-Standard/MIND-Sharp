**mind-csharp** is the official C# reference implementation of the [MIND data standard](https://github.com/SilicoLabs/MIND-Standard).

This SDK provides tools, data structures, and validation logic for representing and serialising multimodal behavioural and neural data in accordance with the MIND specification.
